Hardware Setup
  1. Project pic32mk_gp_db.X
    • Hardware Used
      • PIC32MK GP Development Kit
    • Hardware Setup
      • Connect a micro USB cable to the USB to UART port J25. This will enumerate as a COM port on the PC.
      • Connect a micro USB cable to the USB DEBUG port J12

 

  1. Project pic32mk_mcm_curiosity_pro.X
    • Hardware Used
      • PIC32MK MCM Curiosity Pro Development Board
    • Hardware Setup
      • Connect a micro USB cable to the USB to UART port J400. This will enumerate as a COM port on the PC.
      • Connect a micro USB cable to the USB DEBUG port J500

 

  1. Project pic32mz_das_sk.X
    • Hardware Used:
      • PIC32MZ Embedded Graphics with Stacked DRAM (DA) Starter Kit (Crypto)
    • Hardware Setup:
      • Connect a micro USB cable to the UART-USB port J5
      • For programming, connect a micro USB cable to the USB Debug port J19

 

  1. Project pic32mz_ef_sk.X.
    • Hardware Used:
      • PIC32MZ Embedded Connectivity with FPU (EF) Starter Kit
    • Hardware Setup:
      • Connect a mini USB cable to the UART-USB port J11
      • For programming, connect a mini USB cable to the USB Debug port J3

 

  1. Project sam_e54_xpro.X.
    • Hardware Used:
      • SAM E54 Xplained Pro Evaluation Kit
    • Hardware Setup
      • No Special hardware Setup Required
