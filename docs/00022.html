Hardware Setup

Note: The I2C slave under the Hardware Setup section, refers to the evaluation kit running either the bootloader. 

 

  1. Project sam_c21n_xpro.X
    • Hardware Used
      • SAM C21N Xplained Pro Evaluation Kit
    • Hardware Setup
      • Connect the I2C SDA line of the I2C slave to Pin 11 (SERCOM5 PAD[0]/PB16) on EXT1 of SAM C21N Xplained Pro Evaluation Kit
      • Connect the I2C SCL line of the I2C slave to Pin 12 (SERCOM5 PAD[1]/PB17) on EXT1 of SAM C21N Xplained Pro Evaluation Kit
      • Connect a ground wire between the SAM C21N Xplained Pro Evaluation Kit and the I2C slave
      • Connect a micro USB cable to the DEBUG USB port
