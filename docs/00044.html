Hardware Setup

Note: The I2C host under the Hardware Setup section, refers to the evaluation kit used as the embedded host running the bootloader_host_nvm application. 

 

  1. Project sam_e54_xpro.X
  • Hardware Used
    • SAM E54 Xplained Pro Evaluation Kit
    • Hardware Setup
      • Connect the I2C SDA line of the I2C host to Pin 11 (SERCOM3 PAD[0]/PA22) on EXT1 of SAM E54 Xplained Pro Evaluation Kit
      • Connect the I2C SCL line of the I2C host to Pin 12 (SERCOM3 PAD[1]/PA23) on EXT1 of SAM E54 Xplained Pro Evaluation Kit
      • Connect a ground wire between the SAM E54 Xplained Pro Evaluation Kit and the I2C host
      • Connect a micro USB cable to the DEBUG USB port
