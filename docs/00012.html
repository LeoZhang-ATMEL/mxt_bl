Bootloader Linker Scripts

Bootloader library uses a custom linker script generated through MHC

  • The values populated in the linker script are based on the MHC configurations done for the Bootloader component

 

Bootloader Linker Script for SAM Devices

  • Bootloader is configured to run from RAM in this linker script to allow for self upgrade of the bootloader firmware.
