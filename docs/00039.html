Running The Application

Perform the steps provided under Running The Application section of either the bootloader_host_nvm or bootloader_host_sdcard application to program the test application (if not already done).

  1. Open the Terminal application (Ex.:Tera Term) on the computer.
  2. Configure the serial port settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. If the application is programmed successfully then LED should start blinking and the following message should be printed on the console.
