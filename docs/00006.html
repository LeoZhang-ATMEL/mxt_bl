Applications Help

This section provides help for the I2C Bootloader applications.

I2C Bootloader
Name 
Description 
bootloader 
This example application shows how to use the Bootloader Library to bootload an application using I2C protocol 
bootloader_host_nvm 
This application provides an example of an embedded I2C host to program the user application which is stored in the embedded host's internal flash memory 
bootloader_host_sdcard 
This application provides an example of an embedded I2C host to program the user application which is stored in an SD card on the embedded host. 
test_app 
This example application is used to demonstrate how it can be loaded from bootloader and trigger bootloader from firmware 
I2C Fail Safe Bootloader
Name 
Description 
bootloader 
This example application shows how to use the Bootloader Library to bootload an application on device having dual flash bank feature using I2C protocol 
bootloader_host_nvm 
This application provides an example of an embedded I2C host to program the user application which is stored in the embedded host's internal flash memory using the fail safe bootloader running on the target 
test_app 
This example application is used to demonstrate how it can be loaded from bootloader and trigger bootloader from firmware
